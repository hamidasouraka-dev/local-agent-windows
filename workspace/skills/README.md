# Skills (consignes perso pour l’agent)

Crée un fichier **`.md` par skill** (ex. `mon-style.md`, `cours-droit.md`).

Exemple de contenu :

```markdown
# Mon style
- Tutoiement, français, phrases courtes.
- Toujours proposer un plan avant un long texte.

# Dissertation
- Norme APA pour les références.
```

Dis à l’agent : *« lis le skill skills/mon-style.md »* ou il peut le faire tout seul si la tâche correspond.
