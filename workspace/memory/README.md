# Mémoire persistante (ton disque)

- **`journal.md`** : rempli automatiquement par l’agent avec l’outil `append_memory_note` (entrées datées).
- Tu peux aussi créer d’autres fichiers ici (`.md`, `.txt`) et demander à l’agent de les lire avec `read_file`.

Rien n’est envoyé vers un « cloud mémoire » : tout est dans ce dossier sur ton PC.
